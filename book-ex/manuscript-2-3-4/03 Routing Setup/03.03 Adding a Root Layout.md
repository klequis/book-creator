## 3.3 Adding a Root Layout

Router requires a component as the value of its `root` property. All routes will be rendered as children of the component assigned to `root` so it must render `props.children`. Given it wraps all routes, it is an ideal place to use as your application's top-most or root layout and thus it is referred to as the *root layout*.

1. Create `src/root-layout.jsx`:

{title: "root-layout.jsx"}
```jsx
export default function RootLayout(props) {
  return <>{props.children}</>
}
```

In `app.jsx`:

1. Import the `RootLayout`.
2. Add `RootLayout` as the value of the `<Router>` component's `root` property:

{title: "app.jsx"}

```jsx
import { Router } from "@solidjs/router";
import { FileRoutes } from '@solidjs/start/router'
import RootLayout from './root-layout'
import './app.css'

export default function App() {
  return (
    <div class="app">
      <div class="filename-lg">app.jsx</div>
      <Router root={RootLayout}>
        <FileRoutes />
      </Router>
    </div>
  )
}
```
